# OuterCyrex の 数独谜题

------

算是**前端**的一个小练习题。从7月28日到7月31日正式完工。

主要是在界面的实现上，因为`javascript`学的不够扎实导致出了很多岔子。

网页主要是实现了一个简短的在线数独生成和答题游戏。

加入了我自己的一些想法，比如防止写不出来所以加了个直接生成答案，防止太难所以加入了可以检测格子填的是否正确的功能（有些作弊的感觉）。

------

### 数独该怎么玩？

**数独**要求**每行每列与每个宫**都必须出现`1-9`所有数字且不能出现重复。
